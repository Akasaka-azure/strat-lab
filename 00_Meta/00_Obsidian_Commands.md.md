# Obsidian 運用コマンド集 (00_Obsidian_Commands)

## 1. 基本操作（スピード重視）
* **新規ノート作成**: `Ctrl + N`
    * まずは `01_Inbox` か `10_Daily` に放り込む際に使用。
* **クイック切り替え**: `Ctrl + O`
    * 既存の「結晶化ノート（40_Permanent）」を呼び出す際に必須。
* **コマンドパレット**: `Ctrl + P`
    * 全ての機能の入り口。迷ったらこれ。

## 2. 運用フローを支えるコマンド
* **プロパティの編集**: `Add File Property` ※Ctrl + P
    * ステータス（未整理 / 整理中 / 結晶化済）を管理。
* **テンプレート挿入**: `Insert Template`
    * `00_Meta` 内に作った「日報型」や「仕様書型」を呼び出す。
* **キャンバス作成**: `New Canvas`
    * `80_Maps` で全体構造を視覚化する際に使用。

## 3. 連携・昇華コマンド
* **ファイル移動**: `Move file to another folder`
    * `01_Inbox` から `40_Permanent` へ情報を昇華させる際に多用。
* **リンク作成**: `[[ファイル名]]`
    * 断片（20_Fleeting）同士を繋ぎ、知恵の結晶を作る。
* **PDF/画像のエクスポート**:
    * `10_Docs` へ資料として渡す際に使用。
